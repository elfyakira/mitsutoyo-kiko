## 撮影の基本方針

### 全体のトーン

| 項目 | 方針 |
|-----|------|
| 雰囲気 | 真剣さと温かさの両立。「技術力のある会社」かつ「人の温かい会社」 |
| 作り込み | 過度なポージングは避け、自然な雰囲気を重視 |
| 光 | 工場の照明を活かしつつ、補助光で明るく |
| 色調 | ブルーグレーの工場色をベースに、人物は温かく |
| 清潔感 | 作業着、機械、工場内を清掃した状態で撮影 |

### 避けるべきこと

- スタジオ撮影のような作り込んだ構図
- 硬い表情、緊張した姿勢
- 暗すぎる写真（工場の雰囲気を出しつつも、明るさを確保）
- 過度な加工、フィルター
- 個人情報が写り込む書類等

## 撮影カテゴリ別 詳細指示

---

### カテゴリ1: メインビジュアル用

#### 1-1. トップページ ファーストビュー

**使用箇所**: トップページのメインビジュアル

**撮影内容**:
- 工場内で職人が金型を加工している作業風景
- 機械の操作パネルを見ながら作業している姿
- または、複数の機械が並ぶ工場全景

**構図**:
- 横長（アスペクト比 16:9 推奨）
- 広角で空間の広がりを感じさせる
- 作業者の手元と機械全体が入る

**光・色調**:
- 工場の照明を活かした自然な光
- 機械の金属光沢が出るように
- 暗すぎず、明るさを確保

**必要枚数**: 3-5枚（バリエーション）

**ファイル名例**: `hero_factory_01.jpg`

---

#### 1-2. 会社概要ページ ヘッダー

**使用箇所**: 会社概要ページのページヘッダー

**撮影内容**:
- 工場外観の全景
- 看板と建物が入った構図

**構図**:
- 横長（アスペクト比 16:9 または 21:9）
- 正面やや斜めから
- 建物全体が入るように

**時間帯**:
- 午前中の柔らかい光
- 晴れの日

**必要枚数**: 2-3枚

**ファイル名例**: `building_exterior_01.jpg`

---

#### 1-3. 製品情報ページ ヘッダー

**使用箇所**: 製品情報ページのページヘッダー

**撮影内容**:
- 様々な冷間圧造工具が整然と並んでいる写真
- パンチ、ダイス、カッター等のバリエーション

**構図**:
- 横長
- 俯瞰気味、製品の精密さが伝わる
- 背景はグレーまたは作業台

**光**:
- 金属の光沢が出るように
- 影が強すぎないように

**必要枚数**: 2-3枚

**ファイル名例**: `products_lineup_01.jpg`

---

#### 1-4. 設備紹介ページ ヘッダー

**使用箇所**: 設備紹介ページのページヘッダー

**撮影内容**:
- 工場内の全景、設備が並ぶ様子
- マシニングセンタや研削盤などが整然と配置された風景

**構図**:
- やや高い位置から見下ろす形
- 工場の広さと設備の多さが伝わる

**必要枚数**: 2-3枚

**ファイル名例**: `equipment_overview_01.jpg`

---

#### 1-5. 採用情報ページ ヘッダー

**使用箇所**: 採用情報ページのページヘッダー

**撮影内容**:
- 若手社員3-4名が工場内で笑顔で話している自然なシーン
- 休憩時間や作業の合間の雰囲気

**構図**:
- 横長
- 人物のバストアップ〜全身
- 会話している自然な姿

**表情**:
- 自然な笑顔、作り込まない
- リラックスした雰囲気

**必要枚数**: 3-5枚

**ファイル名例**: `recruit_hero_01.jpg`

### カテゴリ2: 人物写真

#### 2-1. 代表取締役 ポートレート

**使用箇所**: 会社概要ページ 代表メッセージ

**撮影内容**:
- 代表取締役 舟橋佳孝氏のポートレート
- バストアップ

**服装**:
- スーツまたは作業着（会社の雰囲気に合わせて相談）

**背景**:
- 工場内または事務所
- その場の雰囲気が伝わる場所

**表情**:
- 穏やかな笑顔
- 真摯さが伝わる

**構図**:
- アスペクト比 4:5（縦長）
- やや斜めから、威圧感を与えない

**必要枚数**: 5-10枚（表情・角度のバリエーション）

**ファイル名例**: `ceo_portrait_01.jpg`

---

#### 2-2. 社員インタビュー用 ポートレート

**使用箇所**: 採用情報ページ 社員インタビュー

**対象者**:
1. 若手社員（入社2-3年目、20代）
2. 中堅社員（入社5-10年目、30代）
3. （可能であれば）ベテラン社員（入社15年以上）

**撮影内容**:
- 各人物のポートレート
- 作業中の姿
- 笑顔で話している姿

**服装**:
- 作業着（清潔感のあるもの）

**背景**:
- 工場内、各自の持ち場
- 機械の前など

**表情**:
- 自然な笑顔
- 真剣な作業中の表情

**構図**:
- アスペクト比 5:6（縦長）
- バストアップ〜上半身

**必要枚数（1人あたり）**: 10-15枚

**ファイル名例**: `employee_interview_01_portrait.jpg`、`employee_interview_01_working.jpg`

---

#### 2-3. 社員集合写真

## 撮影スケジュール目安

| 撮影日 | カテゴリ | 所要時間 |
|-------|---------|---------|
| 1日目 午前 | 工場外観、設備全景 | 2時間 |
| 1日目 午後 | 設備個別、作業風景 | 3時間 |
| 2日目 午前 | 製品写真 | 3時間 |
| 2日目 午後 | 人物写真（代表、社員） | 3時間 |

## 納品形式

| 項目 | 仕様 |
|-----|------|
| ファイル形式 | JPEG（高品質）およびRAW |
| 解像度 | 最低3000px以上（長辺） |
| カラープロファイル | sRGB |
| 納品方法 | クラウドストレージまたは外付けHDD |
| ファイル命名規則 | `カテゴリ_内容_連番.jpg`（例: `product_punch_01.jpg`） |

## 撮影前の準備依頼

### 企業様へのお願い

1. **清掃**: 撮影日までに工場内、設備、作業台の清掃をお願いします
2. **作業着**: 撮影対象者は清潔な作業着を着用してください
3. **製品準備**: 撮影する製品を事前に選定し、清掃しておいてください
4. **社員への連絡**: 撮影に参加する社員に事前に日程と内容を伝えてください
5. **安全確認**: 撮影に支障のないエリア、稼働可能な設備を事前に確認してください
