#### ■ コンバージョンファネル設計

**ターゲット①: 冷間圧造業界の技術者・購買担当者（30-50代男性）**

```
認知 → 興味 → 比較検討 → 行動 → コンバージョン
  ↓      ↓         ↓        ↓          ↓
検索流入  強み理解   製品・設備  資料請求    問い合わせ
         実績確認   詳細確認   見積依頼    送信完了
```

- **認知段階**:
  - 流入経路: Google検索「冷間圧造 金型」「圧造工具 メーカー」「冷間鍛造 パンチ」
  - 最初に見るページ: トップページ（65%）、製品情報ページ（25%）、会社概要（10%）
  - この段階での訴求: 「創業60年、内製比率92%の冷間圧造工具専門メーカー」
  - 離脱防止策: ファーストビューで「冷間圧造工具専門」「内製比率92%」を3秒以内に伝える

- **興味段階**:
  - 見るページ: 会社概要、設備紹介、製品情報
  - 滞在時間目標: 各ページ2分以上
  - この段階での訴求: 「独自製法ダブルヘックスで金型寿命を延ばす」
  - 次のアクションへの誘導: 「設備を詳しく見る」「製品ラインナップを見る」で詳細ページへ

- **比較検討段階**:
  - 見るページ: 製品詳細、設備紹介、技術力紹介
  - 競合と比較される要素: 製品精度、納期対応力、価格
  - この段階での訴求: 「設計から仕上げまで一貫生産。だから品質が安定し、納期も守れる」
  - 不安解消コンテンツ: 設備一覧（具体的な機械名・台数）、品質管理体制の説明

- **行動段階**:
  - 最後のハードル: 「本当にうちの要望に応えられるのか」「見積りだけでも取れるのか」
  - 背中を押す要素: 「まずはお気軽にご相談ください」「技術的なご質問も歓迎」
  - CTAの配置: トップページ下部、製品ページ下部、各ページフッター上

- **コンバージョン**:
  - 目標行動: お問い合わせフォーム送信
  - フォーム到達率目標: 5%
  - フォーム完了率目標: 70%
  - 離脱防止策: 項目数7個以内、入力時間3分以内、必須項目は最小限

- 開示情報:
  - 社員インタビュー（入社のきっかけ、仕事のやりがい、職場の雰囲気）
  - 1日の流れ（朝8時出社〜17時退社の具体的なスケジュール）

**ステップ3: 募集要項を見た時（ハードル: やや高）**
- 開示情報:
  - 募集職種と仕事内容（具体的に）

#### ■ 「入りたい」と思わせる感情設計

**社員インタビューの設計**:

- 目標感情: 「この人たちと一緒に働きたい」「自分もこうなれそう」

- インタビュー対象者の選定:
  1. 若手社員（入社2-3年目）: 求職者が自分を投影しやすい
  2. 中堅社員（入社5-10年目）: キャリアの見通しを見せる
  3. ベテラン社員（入社15年以上）: 長く働ける安心感

- 質問項目（実際の回答を引き出す）:
  1. 入社のきっかけは？
  2. 入社前と入社後でギャップはあった？
  3. 仕事で一番やりがいを感じる瞬間は？
  4. 職場の雰囲気を一言で表すと？
  5. 休日は何をして過ごしている？
  6. これから入社する人へのメッセージ

- 文言の工夫:
  - NG例: 「やりがいのある仕事です」（抽象的）
  - OK例: 「自分が作った金型で製品が出来上がって、お客様から『これ、調子いいね』って言われた時は、本当に嬉しかったです」（具体的、感情が伝わる）

- 写真の工夫:
  - NG例: 会議室でスーツを着たポートレート
  - OK例: 作業着で機械の前に立つ自然な表情、休憩室で談笑するシーン

### 1-3. ブランド体験設計（👤 専門家③の視点）

#### ■ ブランドの「人格」定義

**もしこの会社が人だったら...**

| 項目 | 定義 |
|-----|-----|
| 年齢 | 50代（60年の歴史を持つ熟練、でも現役で第一線） |
| 性格 | 誠実、寡黙だが芯がある、技術に対する誇り、面倒見が良い |
| 話し方 | 飾らない言葉、専門用語は使うが噛み砕いて説明する、大げさなことは言わない |
| 印象 | 「この人になら任せられる」「腕は確か」「信頼できる」 |
| 服装 | 清潔な作業着、きちんと手入れされた工具 |

**この人格をデザインに落とし込む**:

| 要素 | 表現方法 |
|-----|---------|
| 色 | 紺色（誠実さ、技術への誇り）+ オレンジ（ものづくりの熱量、人の温かさ） |
| フォント | ゴシック体（現代的で読みやすい、堅すぎない） |
| 写真 | 作り込まれたスタジオ写真は避け、現場の自然光で撮影した写真 |
| 余白 | 適度な余白で「品」を出す、詰め込みすぎない |
| トーン | 大げさな煽り文句は使わない、事実ベースの落ち着いた表現 |

- 心理効果: 信頼、誠実、安定、技術力、プロフェッショナル

#### ■ 余白とリズムの設計

**余白の基準値**:

| セクション間 | 120px | 80px | 大きく区切り、セクションの独立性を明確に |
| 見出しと本文 | 40px | 28px | 見出しの重要性を強調 |
| 段落間 | 24px | 20px | 読みやすいリズム |
| リスト項目間 | 16px | 14px | 関連性を保ちつつ区切り |
| 画像とキャプション | 12px | 10px | 密接な関係を示す |

#### ■ 反AIデザインチェックリスト

実装時に必ず確認すること:

**カード使用について**:
- [ ] 本当にカードが必要か？余白とタイポグラフィで表現できないか確認した
- [ ] カードを使う場合、影は控えめ（rgba(0,0,0,0.05)程度）または影なし
- [ ] 全てのカードが同じスタイルになっていない（バリエーションをつける）

**色使いについて**:
- [ ] 紫系の色を使っていない
- [ ] グラデーションを使っていない（または同系色の微細なもののみ）
- [ ] アクセントカラーは「ここぞ」という場所にのみ使用

**装飾について**:
- [ ] 見出しに過剰な装飾（下線、背景色、記号）をつけていない
- [ ] セクション背景色を意味なく交互に変えていない
- [ ] アイコンを過剰に使っていない

**余白について**:
- [ ] 全ての余白が同じ値になっていない
- [ ] 関連する要素は近く、関連しない要素は遠くなっている

**アニメーションについて**:
- [ ] 全ての要素にスクロール連動アニメーションをつけていない
- [ ] アニメーションは一部の重要な要素のみに使用
- [ ] アニメーション時間が画一的（全て0.3秒など）になっていない

**レイアウトについて**:
- [ ] 全てのセクションが同じパターン（見出し→リード→カード→CTA）になっていない
- [ ] 時には非対称のレイアウトを使っている
- [ ] 画像サイズにバリエーションがある
