## コーディングルール

- **globals.cssの@themeブロック内のブランドカラーのみ変更する**
  - `--color-navy` → #1A3A5C に変更
  - `--color-navy-dark` → #0F2840 に変更
  - `--color-accent` → #E65525 に変更
  - `--color-accent-dark` → #CC4A1F に変更
  - ※変数名は変えない（ユーティリティクラスが参照しているため）
- **それ以外のglobals.cssは編集しない**
- **Tailwind CSSを使用** - スタイルはユーティリティクラスで記述

## 最初にやること（セットアップ）

**※すでにdocフォルダとHANDOFF.mdが存在する場合（構成案プロンプトで作成済み）:**

```bash
# 1. docフォルダとHANDOFFを一時退避
cd /mnt/c/client_hp/mitsutoyo-kiko
mv doc ../mitsutoyo-kiko_doc_backup
mv HANDOFF.md ../mitsutoyo-kiko_HANDOFF_backup.md

# 2. 一度親ディレクトリに移動してディレクトリを削除
cd /mnt/c/client_hp
rm -rf mitsutoyo-kiko

# 3. テンプレートをクローン
gh repo clone tenchan000517/template-standard mitsutoyo-kiko

# 4. docフォルダとHANDOFFを戻す
mv mitsutoyo-kiko_doc_backup mitsutoyo-kiko/doc
mv mitsutoyo-kiko_HANDOFF_backup.md mitsutoyo-kiko/HANDOFF.md

# 5. 依存関係インストール
cd mitsutoyo-kiko
npm install
```
